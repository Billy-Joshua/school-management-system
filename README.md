# School Management System PHP + MySQL

Version: 1.0.0

This repository is a school management system built with PHP, MySQL and Bootstrap.

Seeded accounts (use these for first login):
How to change the Admin username and password

1. Generate a bcrypt hash for the new password using the local PHP binary:

```powershell
D:\XAMPP\php\php.exe -r "echo password_hash('NEW_PASSWORD', PASSWORD_DEFAULT);"
```

2. Update the `admin` table using phpMyAdmin SQL tab or the MySQL CLI (replace values accordingly):

```sql
UPDATE `admin` SET `username` = 'new_username' WHERE `admin_id` = 1;
UPDATE `admin` SET `password` = '$2y$10$...generated_hash...' WHERE `admin_id` = 1;
```

Quick add user (via phpMyAdmin)

- Go to `sms_db` → appropriate table (`admin`, `teachers`, `students`, `registrar_office`) → Insert.
- For `password`, paste a value generated by `password_hash` (see command above).

Files of interest

- Login form: `login.php`
- Login handler: `req/login.php`
- Add user forms: `admin/teacher-add.php`, `admin/student-add.php`, `admin/registrar-office-add.php`, `RegistrarOffice/student-add.php`
- Request handlers: files inside `req/` (e.g., `req/student-add.php`, `req/login.php`)

Security notes

- The seeded passwords are simple defaults for demo purposes — change them after first login.
- Keep `DB_connection.php` credentials secure and do not expose them publicly.
